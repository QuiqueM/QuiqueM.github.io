Ejemplo de Enlases

Tenemos 3 tipos diferentes de enlaces en html:

Enlace Nulo:

Este enlace no te lleva a ningun lado,este enlace no se cuenta Enlace Nulo

Enlace Absoluto:

Este enlace te direcciona a otra pagina utilizando la URL completa del sitio. Ir a Google

Enlace Relativo:

Este es un enlace a un sitio local Ejemplos de Estructura HTML

Enlace de Descarga:

En este tipo de enlace el fichero a descargar puede ser local o desde algun sitio en internet, utilizamos el atributo "download" temario del curso
