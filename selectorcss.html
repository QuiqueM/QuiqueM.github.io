Selector de Tipo

Selelecciona un tipo de de elemento html y aplica elestilo a todos los elementos de ese tipo.

<p>{text-aling: justify;}

Selector Descendente

Aplica el estilo a elementos que estan contenidos dentro de otros elementos

p b{color:red} ul li{color:green;}

Selector Hijo

Similar al decendente pero mas restrictivo, aplica solamente a los elementos que sean hijos directos de otro elemento

p>b{color:blue;}

Selector por ID

Un ID es un atributo de un elemento htmly se utiliza para aplicar un estilo de manera particular a un solo elemento

<p id="ejemplo"> un Selector 'id' utliza el siembolo gato #ejemplo{color:blue;}

<p class="ejemplo">parrafo 1</p> <p class="ejemplo">parrafo 2</p> para css utilizar el simbolo punto. .ejemplo{color:red;}
