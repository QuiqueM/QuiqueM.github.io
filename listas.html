Listas

En html nos podemos encontar con 3 tipos de listas, cada lista esta compuesta de elementos, los elementos se especifican por la etiqueta <li>

Las listas ordenadas se declaran con las etiquetas <ol> y </ol>.

Listas Ordenadas

  1. Pimer Elemento
  2. segundo Elemento
  3. tercer Elemento

Las listas no ordenadas se declaran con la etiqueta &ly;ul> y </ul>

Listas NO Ordenadas

  • Pimer Elemento
  • segundo Elemento
  • tercer Elemento

Las listas de Descripcion se declaran por las etiquetas <dl> y </dl>, pero a diferencia de las dos anteriores utiliza las ssigientes etiquetas para sus elementos, <dt> y </dt> para el objeto a describir y <dd> y </dd> para la descripcion del objeto.
